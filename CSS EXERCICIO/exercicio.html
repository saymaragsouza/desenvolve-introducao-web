Título da página

Esse parágrafo descreve o resumo do conteúdo presente nesta página HTML. Por isso ele possui um estilo diferente dos demais parágrafos da página. Para mais informações, Clique!

Antes de 1996, a web era basicamente texto bruto com imagens. Os sites eram feitos apenas em HTML, o que tornava difícil criar layouts mais bonitos e organizados. Quem queria mudar uma cor ou fonte precisava usar tags HTML específicas, o que deixava o código confuso e repetitivo. Em 1994, o norueguês Håkon Wium Lie propôs o conceito de uma linguagem que cuidasse apenas do estilo das páginas. Em 1996, essa ideia virou realidade com o lançamento oficial do CSS1, padronizado pelo W3C (World Wide Web Consortium).

CSS (Cascading Style Sheets) significa Folhas de Estilo em Cascata. É uma linguagem usada para estilizar páginas web. Enquanto o HTML organiza o conteúdo (títulos, parágrafos, imagens), o CSS deixa tudo bonito: muda cores, fontes, tamanhos, posicionamento, espaçamentos, animações e muito mais.
